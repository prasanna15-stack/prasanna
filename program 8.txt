 Create a MongoDB Student Database 

Step 1: Start the process
Step 2: Open the command palette (Ctrl+Shift+P)
Step 3: Search for "MongoDB: Create MongoDB Playground" and press Enter.
Step 4: Delete the default template and type the code to store student database
Step 5: Run the MongoDB Playground (Ctrl + Alt + R)
